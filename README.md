
Background
==========
get ipv4 address for complex bridge zones without dhclient

Dependencies
============
* arping
* arpdig
* zsh

Features
========
* sniff a little bit the network with tcpdump
* prefix generated from Proto: IPv4 (0x0800)
* router lookup from Proto: OSPFv2
* static DNS list

Platform
========
* FreeBSD

Usage
=====
    # ./ipv4.sh
